No Camera Link camera is enumerated after running the Software.

Question

What should I if no Camera Link camera is enumerated after running the Software?

Possible Causes

  • Cause 1: The camera is not properly started or the Camera Link wire line not properly connected to the camera.

  • Cause 2: Third-party frame grabber exception.

Solutions

  • Check if the LED indicator of the camera and the indicator of the third-party frame grabber is in normal status.

  • Reinstall the third-party frame grabber. If it still not works, please contact the manufacturer of the frame grabber.
