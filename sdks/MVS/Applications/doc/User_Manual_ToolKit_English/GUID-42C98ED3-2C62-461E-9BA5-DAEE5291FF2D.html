V4.4.0 and Earlier Versions

This section introduces the new features, updates, and improvements of V4.4.0 and earlier versions.

V4.4.0

This version was released in August 2024.

Note

The following only introduces the new features and optimizations in this version. Those not mentioned only iterate with the client and have no new or updated features.

Firmware Upgrade Tool

  • Changed Camera Link interface to Serial Port and GenTL to PCIe.

  • Added Fuzzy Matching to frame grabber upgrade page. See Upgrade a Frame Grabber.

Feature Import/Export Tool

Changed Camera Link interface to Serial Port and GenTL to PCIe.

Port Editing Tool

Changed Camera Link interface to Serial Port and GenTL to PCIe.

Event Configuration Tool

  • Adds Event Monitor in the main UI for quick event viewing. For details, see Configure the Event.

  • Supports copying node name through keyboard shortcut. For details, see Configure the Event.

  • Adds shortcut entry to Event monitor. See Event Configuration Tool.

PCIe Diagnostic Tool

Supports XoFLink frame grabber. See PCIe Diagnostics Tool.

Device Diagnostic Tool

Adds Stop when occur and Timer period type in exception IO locating. See IO WaveForm Detection.

V4.3.0

This version was released in October 2023.

Note

The following only introduces the new features and optimizations in this version. Those not mentioned only iterate with the client and have no new or updated features.

Firmware Upgrade Tool

Optimized the operations of upgrading and configuring frame grabbers. See Upgrade a Frame Grabber.

Feature Import/Export Tool

Supports importing and exporting parameters of frame grabbers. For details, refer to Import/Export Features.

Event Configuration Tool

Added the function of importing/exporting events.

ISP Tool

  • Adds the function of quick correction.

  • Adds the button of hue and saturation in the image operation area.

  • Supports editing imported images.

  • Supports manually drawing a curve line.

  • Updates the parameters of CCM.

  • LUT supports setting the LUT line type.

  • Adds the algorithm of local contrast enhancement.

  • Embeds the ISP tool in the MVS Software.

V4.2.0

This version was released in June 2023.

Note

The following only introduces the new features and optimizations in this version. Those not mentioned only iterate with the client and have no new or updated features.

Firmware Upgrade Tool

You can use the Firmware Upgrade Tool to configure firmware upgrading settings and upgrade the firmware of cameras and frame grabbers.

Feature Import/Export Tool

Provides the Feature Import/Export Tool to import/export the device parameters, etc.

Port Editing Tool

Provides the Port Editing Tool to edit the frame grabber's port.

Event Configuration Tool

Provided reference for viewing and configuring frame grabbers' events.

PC Diagnostic Tool

Provides reference for checking the performance of the PC system, drivers, and software environment.

PCIe Diagnostic Tool

Provides PCIe diagnostic tool to check the PCIe running status of frame grabbers.

Device Diagnostic Tool

This is the first version of Device Diagnostic Tool. The following are the main features.

  • Supports diagnostic event overview. See Event Overview.

  • Supports detailed event diagnosis. See Event Diagnostic.

  • Supports viewing IO waveforms. See IO WaveForm Detection.

  • Supports log management. See Log Management.

V1.2.0

This version was released in March 2023.

Note

The following is the new features and updates of the ISP Tool.

ISP Tool

  • Supports Bayer interpolation method. See General Settings.

  • Supports the white balance algorithm. See White Balance for details.

  • Supports Gamma algorithm. See Gamma for details.

  • Supports exporting denoise calibration file. See Denoise for details.

  • Supports CCM calibration. See CCM for details.

V1.1.0

This version was released in September 2021.

Note

The following is the new features and updates of the ISP Tool.

ISP Tool

  • Updated the running environment, see System Requirements.

  • Added General Settings, Video And Picture Settings, Tool, and Help on the menu bar. See Menu Bar.

  • Added the LSC (Lens Shading Correction) algorithm. See Lens Shading Correction (LSC).

  • Added the DDS algorithm. See DDS.

  • The SixAixs algorithm is renamed as Super Palette. See Super Palette.

  • Added the purple fringe correction algorithm. See Purple Fringe.

  • Updated the parameters of sharpness. See Sharpness.

  • Added the super binning algorithm. See Super Binning.

V1.0.0

This version was released in March 2021.

Note

The following is the new features and updates of the ISP Tool.

ISP Tool

Supports online and offline modes, which allows you to process images from local paths or acquired from cameras.
