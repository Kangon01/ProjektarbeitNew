View and Set Camera Feature Tree

You can open and configure the feature tree of a connected camera.

Click to show the Camera List. In the Camera List, right-click the connected camera and select Feature Tree to show the feature tree window where you can view and configure the camera features.

Figure 1 Feature Tree
Note

  • Select a camera feature, the information about the feature will appear at the bottom.

  • You can search for a camera feature in the search box at the top.
