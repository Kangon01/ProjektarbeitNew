图像增强

图像增强模块包含LUT、对比度、锐化和局部对比度增强算法模块。

说明

图像增强类算法统一对图像第一行不进行处理，此为正常现象。

LUT

LUT是一个可供用户自定义的灰度映射表，可对感兴趣的灰度范围进行拉伸、凸显等操作。

使用LUT模块可直接进行校正，相关参数如下：

Gamma

设置Gamma值可以实现图像的非线性色调映射，参数范围为[0，100]。

对比度

参数范围为[-1，1]，当值为0时，表示对比度保持不变，小于0表示对比度降低，大于0表示对比度提高。

亮度

参数范围为[-1，1]，当值为0时，表示亮度保持不变，小于0表示亮度降低，大于0表示亮度提高。

LUT线型
对LUT线型进行选择，有如下几种线型：
Adaptive

自适应线。根据输入的Gamma、对比度以及亮度数值自适应生成LUT曲线，不可自定义更改。

Fold Line

折线。可自定义更改，双击折线中需要更改的位置，拖动即可。

Curve

曲线。可自定义更改，双击曲线中需要更改的位置，拖动即可。

Free Line
自由线。可自定义绘制，单击界面拖动绘制即可。

对比度

通过提升对比度可使图像更加清晰，色彩鲜明。

使用对比度模块时可直接进行校正，相关参数如下：

对比度调节强度

范围为[1，10000]，数值越大，对比度越强。

图 2 对比度校正参数

锐化

锐化模块可使图像纹理边缘更加清晰。

使用锐化模块时可直接进行校正，相关参数如下：

锐度调节半径

即锐度调节滤波核半径，取值范围为[1，21]。设置半径越大，耗时越长。

锐度调节阈值

用于进行锐化边缘选择的阈值，取值范围为[0，255]。设置阈值越小，锐化处理的像素越多。

锐度调节正向/负向强度

用于控制锐化强度，取值范围为[0，500]。设置数值越大，锐化强度越高。

图 3 锐化校正参数

局部对比度增强

在低光照或光照不均的场景下，通过自适应提高图像的局部对比度，可使图像暗区亮度提高的同时保持亮区的亮度不变。

使用局部对比度模块时可直接进行校正，相关参数如下：

LCE双边滤波窗口

设置双边滤波窗口的大小，参数范围为[3，41]。

LCE高斯空域标准差

设置双边滤波空间域高斯函数的标准差，参数范围为[100，10000]。

LCE高斯值域标准差

设置双边滤波值域高斯函数的标准差，参数范围为[1000，20000]。

LCE Gamma值

设置图像的Gamma值大小，参数范围为[1，10000]。

LCE纹理图像比例系数

设置图像纹理细节的融合比例，数值越大，纹理细节增强越明显，参数范围为[0，1280]

LCE加权曲线对称点

设置图像暗区和亮区的像素灰度值分段点，参数范围为[0，4095]。

LCE提亮控制系数比率

设置图像提亮程度，数值越小，图像提亮越明显，参数范围为[0，256]。
