运行环境

为确保图像调试工具能正常安装及运行，对PC的配置有所要求。

推荐硬件配置

  • CPU：Intel i3-8100T 3.1GHz

  • 内存：8GB

  • 显卡：Intel核芯显卡 630

  • 网卡：Intel千兆网卡

  • USB：USB3.0接口

最低硬件配置

  • CPU：Intel E3845 1.91GHz

  • 内存：4GB

  • 显卡：Intel核芯显卡 Gen7 GPU

  • 网卡：千兆网卡

  • USB：USB2.0接口

软件要求

  • 操作系统：Windows 7/10（32/64位），Windows 11（64位）

  • SDK版本：工业相机SDK V4.2.0及以上版本
