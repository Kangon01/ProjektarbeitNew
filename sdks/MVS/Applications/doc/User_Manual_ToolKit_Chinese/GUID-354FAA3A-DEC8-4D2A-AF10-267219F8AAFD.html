编辑图像

可根据需求对已导入的图片进行编辑操作，仅支持Mono8和RGB24格式的图像。

已导入本地图像或相机图像。

  1. 点击图像操作区域右上角的编辑，打开图像编辑器，如下图所示。
    图 1 图像编辑器
  2. 通过点击图像编辑器上方的按钮可相关操作，如下表所示。
    表 1 编辑图像

    操作

    描述

    撤销：撤回上一步编辑操作。

    重做：重新进行上一步编辑操作。

    还原：清空所有编辑效果。

    裁剪：对图像进行自定义裁剪，具体操作如下图所示。

    1. 单击。

    2. 在图像上调整裁剪区域大小和位置。

    3. 双击界面即可。

    图 2 裁剪

    旋转裁剪：对图像进行自定义角度裁剪，具体操作如下图所示。

    1. 单击。

    2. 在图像上单击并旋转调整出现的十字坐标轴。

    3. 确定角度后再次单击，拖拽出裁剪区域，并调整大小和位置。

    4. 双击界面即可。

    图 3 旋转裁剪

    右旋90度：将图像向右旋转90°。

    左旋90度：将图像向左旋转90°。

    上下翻转：将图像进行上下镜像翻转。

    左右翻转：将图像进行左右镜像翻转。

    网格辅助线。

    优化图片：可将图像宽度优化为3200宽度，以免图片过大或过小。
  3. 根据需求完成相关编辑操作后，单击完成。

    已编辑的图像显示在图像操作区域。

    说明

    原有图像不被覆盖。

  4. 可选操作： 如需保存已编辑的图像，单击保存副本，选择保存路径并保存。
