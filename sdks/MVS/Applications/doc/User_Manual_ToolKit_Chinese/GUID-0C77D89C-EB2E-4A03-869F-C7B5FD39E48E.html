色彩校正

色彩校正模块包括6种算法模块：白平衡、CCM、超级调色盘、CLUT、去紫边、Gamma。

说明

色彩校正仅支持RGB格式的图片输入。当输入图片格式为Bayer时，可先通过色彩插值转换成RGB格式，再进行色彩校正处理。

白平衡

白平衡模块可用于纠正偏色，使不同光源条件下画面呈现的色彩与人眼观看的色彩相近。

使用白平衡模块时可先进行标定再校正，也可省略标定过程，点击直接校正获取校正参数。

标定参数

白平衡需对校正前图像绘制24色卡ROI，且可通过ROI宽度和ROI高度对单个ROI的宽度和高度值进行调整，具体可参考图像降噪章节。还需对以下参数进行设置：

使用输入颜色

  • 不使用：无输入颜色。

  • 原始颜色：点击原始颜色并选择需导入的txt文件，或者点击增加行并选择原始颜色进行添加。

    说明

    • txt文件记录颜色格式的要求：一行代表一个颜色，按照r,g,b的格式写入，不允许有空白行。

    • 输入颜色过程中如需删除，可点击删除行。

标定方法

可选AVG（平均法）、SOLVE（解析法）。

量化系数

可将白平衡矩阵中的数值整数化，便于进行计算。设置后白平衡矩阵中的数值会随之改变。

当校正参数中的量化系数选择1时，矩阵数值可输入浮点数。

CCM

CCM是一个3*3的色彩校正矩阵，可调整RGB中红、绿、蓝三色分量的比例关系。

使用CCM模块时可先进行标定再校正，也可省略标定过程，直接导入校正参数进行图像校正，通过点击直接校正实现。

标定参数

CCM标定支持输入多张标定图像和参考图像，对校正前的标定图像和参考图绘制ROI以提取输入颜色和参考颜色。

说明

图像列表最多支持导入24张图像。

ROI绘制

支持24色卡ROI、140色卡ROI、自定义ROI和自由ROI四种绘制方式，具体介绍可参考图像降噪章节。

说明

对于CCM标定，绘制ROI时预览窗口的图像上会显示ROI序号。

使用输入颜色

  • 不使用：无输入颜色。

  • 原始颜色：点击原始颜色并选择需导入的txt文件，或者点击增加行并选择原始颜色进行添加。

  • 参考颜色：点击参考颜色并选择需导入的txt文件，或者点击增加行并选择参考颜色进行添加。

  • 全部颜色：点击原始颜色或参考颜色可分别导入参考颜色的txt文件，或者点击增加行并选择原始颜色和参考颜色进行添加。

说明

  • txt文件记录颜色格式的要求：一行代表一个颜色，按照r,g,b的格式写入，不允许有空白行；原始颜色和参考颜色应分别存在两个文件中。

  • 原始颜色和参考颜色个数限制均为140，且原始颜色与参考颜色个数需保持一致。

  • 输入颜色过程中如需删除，可点击删除行。

Gamma类型

可根据实际需求进行选择，分为以下4种。

None

不启用Gamma。

User Value

Gamma值

User Curve

通过导入Gamma曲线文件进行控制，相关参数如下：

Gamma曲线

单击导入曲线，选择本地的Gamma曲线文件进行导入。

说明

Gamma曲线文件请联系技术支持获取。

LRGB2SRGB

由算法内部生成Gamma值。

校正参数

CCM校正可对以下参数进行调整：

Gamma类型

设置方法和标定参数中相同。

量化系数

可将CCM矩阵中的数值整数化，便于进行运算。设置后CCM矩阵中的数值会随之改变。

CCM矩阵

单击编辑可修改矩阵数值，从而调整RGB中红、绿、蓝三色分量的比例关系。

说明

当校正参数中的量化系数选择1时，矩阵数值可输入浮点数。

饱和度使能

使能后，饱和度下的滑块才可拖动，且量化系数和CCM矩阵不可编辑。

饱和度

拖动滑块可调整图像整体饱和度，CCM矩阵中的数值会随之改变。

超级调色盘

超级调色盘模块可对图像中的不同颜色区域进行色调与饱和度调节。

使用超级调色盘模块时可先标定再校正，也可省略标定过程，直接导入校正参数进行图像校正，通过点击直接校正实现。

标定参数

超级调色盘标定时需要导入一张参考图，并对校正前图像和参考图绘制ROI，具体方法及相关参数介绍可参考CLUT章节。

校正参数

校正参数处可调节红/绿/蓝/青/品红/黄六种颜色的色调和饱和度，通过拖动滑块调整数值。

使能量化前，色调和饱和度的数值可输入浮点数；使能量化后，仅可输入整数。

图 5 超级调色盘校正参数

CLUT是一个三维的颜色查找表，可以针对特定的颜色进行调整，同时不影响其他颜色。

使用CLUT模块时需要进行标定和校正。

标定参数

CLUT标定需要导入一张参考图，并对校正前图像和参考图绘制ROI，还需对以下参数进行设置：

ROI绘制

使用输入颜色

参数具体介绍请参考CCM章节。

Gamma类型

参数具体介绍请参考CCM章节。

量化系数

可将CLUT矩阵中的数值整数化，便于进行运算。

平滑因子

设置数值大小将影响CLUT校正的范围。

去紫边

去紫边模块可有效处理图像中产生的“色像差”现象，去除由于镜头折射、光线等原因引起的边缘假色，可以有效提高色彩的保真度和图像清晰度。

使用去紫边模块时可单击直接校正获取校正参数，相关参数如下：

滤波核尺寸

即进行紫边校正滤波的滤波核大小。理论上来说，“紫边”越粗，滤波核尺寸的值需设置越大。

边缘阈值

用于判断图像边缘点的阈值，范围为[0，2040]。边缘阈值越小，被判断为边缘的像素点越多，参与紫边校正的纹理边缘越多。

图 8 去紫边校正参数

使用Gamma模块时无需标定，点击直接校正获取校正参数。

校正参数

Gamma类型

可根据实际需求进行选择，分为以下5种：

None

不启用Gamma。

User Value

通过设置Gamma数值进行控制，相关参数如下：

Gamma值

输入Gamma数值，范围为[0.1，3.9]。

User Curve

通过导入Gamma曲线文件进行控制，相关参数如下：

Gamma曲线

单击导入曲线，选择本地的Gamma曲线文件进行导入。

说明

Gamma曲线文件请联系技术支持获取。

LRGB2SRGB

由算法内部生成Gamma值。

SRGB2LRGB

由算法内部生成Gamma值。
